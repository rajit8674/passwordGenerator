## Name & DOB Password Generator

This project generates a password based on user name and date of birth using JavaScript.

## Live -- (https://rajit8674.github.io/passwordGenerator/)

##  Tech Stack
- HTML
- CSS
- JavaScript

##  Features
- Name based password logic
- Date of birth combination
- Random symbol for better strength
- Beginner friendly logic

##  Note
DOB based passwords are not recommended for real security purposes.
